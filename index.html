// 01   OVERVIEW
Introduction
This project detects nearby obstacles using ultrasonic pulses, displays system status on an LCD screen, and uses a servo motor to rotate the sensor to scan different angles. The system works without physical contact and can alert the user when an object enters the sensing area — providing a low-cost, real-time, contactless safety solution.

// 03   HOW IT WORKS
Methodology
INIT

System Startup

Arduino powers up. All components get initialized. LCD shows "Scanning..." and servo moves to starting position.

SCAN

Rotational Scanning

Servo motor rotates the ultrasonic sensor left to right. At each angle, the sensor emits ultrasonic pulses and listens for reflections.

DETECT

Object Detection

Arduino calculates distance from the time delay of returning sound waves. If an object is within threshold, an alert is triggered.

ALERT

Warning Output

LCD shows "WARNING FOREIGN BODY!" and buzzer/LED turn ON. After alert, servo continues scanning in a continuous loop.

CLEAR

Clear Zone

If no object detected, LCD shows "AREA IS EMPTY" — buzzer stays OFF and scanning continues uninterrupted.
